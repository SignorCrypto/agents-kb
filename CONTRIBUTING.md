# Contributing

This is my first open source project.

If you like `Agents-KB` and want to help, I really appreciate the interest.

I am not fully accepting contributions yet, because I am still figuring out the best way to manage them.

I will probably try to rely more on a concept I designed which are the prompt requests. You can read more about that idea here: [Prompt Requests](https://github.com/SignorCrypto/prompt-request).
